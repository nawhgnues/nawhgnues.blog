### 블로그 레포지토리

next.js, shadcn/ui, mdx, vercel 이용한 블로그

### 📂 폴더 구조

```
app
   about --- 자기소개
   blog  --- 블로그
   components --- 컴포넌트
      about
      common
      icon
      mdx
      post_detail
      post_list
      ul
   config
      const.ts
      globals.css
      types.ts
   data --- 자기소개 데이터
   layouts
   hook
   lib
   posts --- 블로그 포스트 데이터
   projects --- 프로젝트 관련 데이터
```

### 🛠️ 기술 스택

- Next.js

- Tailwind CSS + Shadcn/ui

- MDX

- Vercel
